# File: frontend/vite.config.js

## Overview
The configuration file for Vite, the build tool and development server used for the frontend.

## Key Configuration
- **Plugins**: Includes the `@vitejs/plugin-react` to enable React support (JSX, Fast Refresh, etc.).
- **Base Config**: Uses `defineConfig` to provide type-safe configuration.

## Role in the System
Manages the build process, bundling, and local development environment for the React frontend.
